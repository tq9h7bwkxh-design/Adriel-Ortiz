# Soccer Game

## Overview
This is a multiplayer and single-player soccer game built with WebSocket support, a 3D game client using Three.js and React, an AI opponent system, player progression, team management, and match statistics.

## Structure
- **/server**: Contains the main game server.
- **/client**: Contains the 3D game client.
- **README.md**: Project overview and instructions.

## Setup Instructions
1. Clone the repository.
2. Navigate into the `/server` directory and run `npm install` to install server dependencies.
3. Navigate into the `/client` directory and run `npm install` to install client dependencies.
4. Start the server by running `node index.js` in the `/server` directory.
5. Start the client by running `npm start` in the `/client` directory.

## Features
- WebSocket support for real-time multiplayer gaming.
- 3D graphics using Three.js.
- AI opponent provided for single-player mode.
- Player progression and team management.
- Match statistics tracking.
